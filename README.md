# HoaxBuster_CITIzen
Description (≤350 characters): HoaxBuster is an AI-powered tool by Team CITizen from the Great Malaysia AI Hackathon 2025. It combats misinformation by classifying content as Fake, True, Misleading, Uncertain, or Undetected, with credibility scores, fact-check sources, and educational support. Built with AWS, Python, and web tech.
